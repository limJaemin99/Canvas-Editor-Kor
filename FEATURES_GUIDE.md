## 상단 툴바 기능

### 1️⃣ 편집 작업

#### 실행취소 / 재실행
- **실행취소**: 이전 작업 취소
  - 단축키: `Ctrl+Z` (Mac: `⌘+Z`)
- **재실행**: 취소한 작업 다시 실행
  - 단축키: `Ctrl+Y` (Mac: `⌘+Y`)

#### 서식 도구
- **서식 복사**: 선택한 텍스트의 서식을 복사
  - 클릭: 한 번만 적용
  - 더블클릭: 연속 적용 모드 (여러 곳에 붙여넣기 가능)
- **서식 지우기**: 선택한 텍스트의 모든 서식 제거

### 2️⃣ 글꼴 설정

#### 글꼴 (Font Family)
**사용 가능한 글꼴:**
- 맑은 고딕 (Malgun Gothic)
- Microsoft YaHei
- 바탕 (Batang)
- 돋움 (Dotum)
- 굴림 (Gulim)
- 궁서 (Gungsuh)
- Arial
- Segoe UI
- Ink Free
- Fantasy

#### 글꼴 크기 (Font Size)
**사용 가능한 크기:** 6, 7, 8, 10, 12, 14, 16, 18, 20, 21, 24, 29, 32, 34, 48, 56

**크기 조절 버튼:**
- **글꼴 크게**: `Ctrl+[` (Mac: `⌘+[`)
- **글꼴 작게**: `Ctrl+]` (Mac: `⌘+]`)

---

### 3️⃣ 텍스트 스타일

#### 기본 스타일
| 기능 | 단축키 (Windows) | 단축키 (Mac) | 설명 |
|------|-----------------|--------------|------|
| **굵게** | `Ctrl+B` | `⌘+B` | 텍스트를 굵게 |
| **기울임** | `Ctrl+I` | `⌘+I` | 텍스트를 기울임체로 |
| **밑줄** | `Ctrl+U` | `⌘+U` | 텍스트에 밑줄 추가 |
| **취소선** | `Ctrl+Shift+X` | `⌘+Shift+X` | 텍스트에 취소선 추가 |

#### 밑줄 스타일 옵션
- 실선 (Solid)
- 이중선 (Double)
- 점선 (Dashed)
- 점 (Dotted)
- 물결선 (Wavy)

#### 첨자
- **위첨자**: `Ctrl+Shift+,` (Mac: `⌘+Shift+,`)
  - 예: H₂O의 ₂
- **아래첨자**: `Ctrl+Shift+.` (Mac: `⌘+Shift+.`)
  - 예: X²의 ²

#### 색상
- **글꼴 색**: 텍스트 색상 선택 (컬러 피커)
- **강조 색**: 텍스트 배경색 선택 (형광펜 효과)

### 4️⃣ 단락 설정

#### 제목 (Title)
**사용 가능한 제목 레벨:**
- 본문 (일반 텍스트)
- 제목 (26px) - `Ctrl+Alt+1` - 목차에 표시
- 편 (24px) - 가운데 정렬, 목차에 표시
- 장 (22px) - 가운데 정렬, 목차에 표시
- 절 (20px) - 가운데 정렬, 목차에 표시
- 관 (18px) - 가운데 정렬, 목차에 표시
- 조 (16px) - 목차에 표시
- 항 (16px) - 조보다 1단계 들여쓰기, 목차에 미표시
- 호 (16px) - 조보다 2단계 들여쓰기, 목차에 미표시
- 목 (16px) - 조보다 3단계 들여쓰기, 목차에 미표시
- 단 (16px) - 조보다 4단계 들여쓰기, 목차에 미표시

**특징:**
- 제목/편/장/절/관/조는 목차에 표시됨
- 편/장/절/관은 자동으로 가운데 정렬
- 항/호/목/단은 들여쓰기가 자동 적용되며 목차에는 표시되지 않음

#### 정렬 (Alignment)
| 정렬 방식 | 단축키 (Windows) | 단축키 (Mac) | 설명 |
|----------|-----------------|--------------|------|
| **왼쪽 정렬** | `Ctrl+L` | `⌘+L` | 텍스트를 왼쪽 정렬 |
| **가운데 정렬** | `Ctrl+E` | `⌘+E` | 텍스트를 가운데 정렬 |
| **오른쪽 정렬** | `Ctrl+R` | `⌘+R` | 텍스트를 오른쪽 정렬 |
| **양쪽 정렬** | `Ctrl+J` | `⌘+J` | 양쪽 끝을 맞춤 |
| **균등 분할** | `Ctrl+Shift+J` | `⌘+Shift+J` | 글자 간격을 균등하게 |

#### 줄 간격 (Line Spacing)
**사용 가능한 간격:** 1, 1.25, 1.5, 1.75, 2, 2.5, 3

#### 목록 (List)
**단축키:** `Ctrl+Shift+U` (Mac: `⌘+Shift+U`)

**목록 종류:**
1. **목록 취소** - 목록 제거
2. **순서 목록** - 1, 2, 3...
3. **체크박스 목록** - ☑️ 체크 가능한 목록
4. **채운 원 목록** - ● 목록
5. **빈 원 목록** - ○ 목록
6. **빈 사각형 목록** - ☐ 목록

### 5️⃣ 삽입 기능

#### 표 (Table)
- **삽입 방법**: 표 아이콘 클릭 → 행/열 선택 (최대 10×10)
- **기능**: 마우스로 드래그하여 크기 선택

#### 이미지 (Image)
- **지원 형식**: PNG, JPG, JPEG, SVG, GIF
- **삽입 방법**: 파일 선택 → 자동 크기 계산 후 삽입
- **추가 기능**:
  - 우클릭으로 캡션 추가/수정 가능
  - 이미지 번호 자동 지정 (`{imageNo}`)

#### 하이퍼링크 (Hyperlink)
**입력 정보:**
- **텍스트**: 표시될 텍스트
- **링크**: URL 주소

#### 구분선 (Separator)
**구분선 스타일:**
- 실선
- 점선 (작은 간격)
- 점선 (큰 간격)
- 점-대시선
- 점-대시-대시선
- 이중 점-대시선

#### 워터마크 (Watermark)
**추가 옵션:**
- **내용**: 워터마크 텍스트
- **색상**: 워터마크 색상 (기본: #AEB5C0)
- **글꼴 크기**: 크기 설정 (기본: 120)
- **투명도**: 0~1 사이 값 (기본: 0.3)
- **반복**: 반복 여부 선택
  - 반복 시: 수평/수직 간격 설정 가능

**기능:**
- 워터마크 추가
- 워터마크 삭제

#### 페이지 나누기 (Page Break)
- 현재 위치에서 새 페이지 시작

#### 코드 블록 (Code Block)
- **입력**: 코드 입력창 (500×300)
- **기능**: JavaScript 구문 강조 (Prism.js 사용)
- **스타일**: 색상, 굵기, 기울임 자동 적용

### 6️⃣ 컨트롤 (Control)

폼 요소를 삽입하여 대화형 문서 작성

#### 텍스트 컨트롤
**설정 항목:**
- 플레이스홀더
- 기본값

#### 숫자 컨트롤
**설정 항목:**
- 플레이스홀더
- 기본값

#### 목록 컨트롤 (Select)
**설정 항목:**
- 플레이스홀더
- 기본값 (code)
- 값 집합 (JSON 형식)
  ```json
  [
    {"value":"있음", "code":"98175"},
    {"value":"없음", "code":"98176"}
  ]
  ```

#### 날짜 컨트롤
**설정 항목:**
- 플레이스홀더
- 기본값
- 날짜 형식
  - `yyyy-MM-dd hh:mm:ss`
  - `yyyy-MM-dd`

#### 체크박스 컨트롤
**설정 항목:**
- 기본값 (쉼표로 구분하여 여러 값 가능)
- 값 집합 (JSON 형식)

#### 라디오버튼 컨트롤
**설정 항목:**
- 기본값
- 값 집합 (JSON 형식)

---

### 7️⃣ 기타 삽입 요소

#### 체크박스
- 클릭하여 체크/해제 가능한 체크박스 삽입

#### 라디오버튼
- 그룹으로 관리되는 라디오버튼 삽입

#### LaTeX
- 수식 입력창에 LaTeX 문법으로 수식 작성
- 예: `E = mc^2`, `\frac{a}{b}`

#### 날짜
**형식 선택:**
- `yyyy-MM-dd` (예: 2026-02-24)
- `yyyy-MM-dd hh:mm:ss` (예: 2026-02-24 10:30:45)

#### 콘텐츠 블록 (Block)
**블록 종류:**

1. **웹 주소 (iframe)**
   - **주소**: URL 입력
   - **HTML**: HTML 코드 직접 입력
   - **크기**: 너비(선택), 높이(필수)

2. **비디오 (video)**
   - **주소**: 비디오 파일 URL
   - **크기**: 너비(선택), 높이(필수)

---

### 8️⃣ 검색 및 출력

#### 검색 및 바꾸기
**단축키:** `Ctrl+F` (Mac: `⌘+F`)

**기능:**
- **검색**: 텍스트 검색
- **바꾸기**: 검색된 텍스트를 다른 텍스트로 교체
- **이전/다음**: 검색 결과 이동
- **옵션**:
  - 정규식 사용
  - 대소문자 무시
  - 선택 영역 내 검색

**검색 결과 표시:** `1/5` (현재 위치/전체 개수)

#### 인쇄
**단축키:** `Ctrl+P` (Mac: `⌘+P`)
- 문서를 인쇄 미리보기로 출력

---

### 9️⃣ 문서 관리

#### 저장 (보라색 버튼)
- **기능**: 현재 문서를 JSON 형식으로 서버에 저장
- **엔드포인트**: `/lmxmng/webEditor/saveDoc.do`
- **전송 데이터**:
  ```json
  {
    "version": "0.9.127",
    "header": [...],
    "main": [...],
    "footer": [...],
    "options": {...},
    "timestamp": "2026-02-24T10:30:00.000Z"
  }
  ```

#### 예시 (분홍색 버튼)
- **기능**: 예시 데이터 불러오기
- **내용**: 의료 진료 기록 샘플
  - 제1인민병원
  - 외래 진료 기록
  - 환자 정보, 진단 내용 등

#### 초기화 (주황색 버튼)
- **기능**: 문서를 빈 상태로 초기화
- **확인**: 모든 내용이 삭제된다는 경고 표시

## 우클릭 메뉴

문서 내에서 마우스 우클릭 시 표시되는 컨텍스트 메뉴

### 1. 주석 (批注)
**표시 조건:**
- 읽기전용 모드가 아님
- 텍스트가 선택되어 있음
- 본문 영역(MAIN)에서 클릭

**기능:**
- 선택한 텍스트에 주석 추가
- 주석 내용 입력 (textarea)
- 주석 목록에 표시되며 클릭 시 해당 위치로 이동

### 2. 캡션 추가
**표시 조건:**
- 읽기전용 모드가 아님
- 이미지를 선택함
- 이미지에 캡션이 없음

**기능:**
- 이미지에 캡션 추가
- `{imageNo}` 사용 시 이미지 번호 자동 표시

### 3. 캡션 수정
**표시 조건:**
- 읽기전용 모드가 아님
- 이미지를 선택함
- 이미지에 캡션이 있음

**기능:**
- 기존 캡션 내용 수정

### 4. 서명
**표시 조건:**
- 읽기전용 모드가 아님
- 편집기에 포커스가 있음

**기능:**
- 서명 도구 열기
- 마우스/터치로 서명 그리기
- 서명을 이미지로 삽입

### 5. 포맷 정리
**표시 조건:**
- 읽기전용 모드가 아님

**기능:**
- 문서 포맷 자동 정리
- 일관된 스타일 적용

### 6. 낙서 지우기
**표시 조건:**
- 주석(GRAFFITI) 모드인 경우

**기능:**
- 모든 낙서 정보 삭제

## 하단 정보바

문서 편집 중 실시간 정보 표시

### 왼쪽 영역
- **목차**: 목차 패널 열기/닫기
- **페이지 모드**:
  - 분할 (Paging) - 페이지별로 표시
  - 연속 (Continuity) - 스크롤 방식
- **보이는 페이지**: 현재 화면에 보이는 페이지 번호 (예: 1、2、3)
- **페이지**: 현재 페이지/전체 페이지 (예: 1/5)
- **문자 수**: 전체 문자 수
- **줄**: 현재 커서 줄 번호
- **열**: 현재 커서 열 번호

### 중앙
- **편집 모드**: 현재 편집 모드 표시 (클릭하여 변경)

### 오른쪽 영역
- **축소**: 페이지 축소 (`Ctrl+-`)
- **표시 비율**: 현재 배율 (예: 100%) - 클릭 시 100%로 복원
- **확대**: 페이지 확대 (`Ctrl+=`)
- **용지 종류**:
  - A4 (794×1123)
  - A2 (1593×2251)
  - A3 (1125×1593)
  - A5 (565×796)
  - 5호 봉투 (412×488)
  - 6호 봉투 (450×866)
  - 7호 봉투 (609×862)
  - 9호 봉투 (862×1221)
  - 법률 용지 (813×1266)
  - 편지지 (813×1054)
- **용지 방향**:
  - 세로 (Vertical)
  - 가로 (Horizontal)
- **여백**: 위/아래/왼쪽/오른쪽 여백 설정
- **전체 화면**: 전체 화면 모드 토글 (`F11`)
- **편집기 설정**: 편집기 옵션 JSON 편집

## 편집 모드

클릭하여 순환 전환

### 1. 편집 모드 (EDIT)
- 모든 편집 기능 사용 가능
- 기본 모드

### 2. 청결 모드 (CLEAN)
- 불필요한 UI 요소 최소화
- 집중 편집 환경

### 3. 읽기전용 모드 (READONLY)
- 문서 보기만 가능
- 편집 불가
- 검색, 인쇄만 가능

### 4. 양식 모드 (FORM)
- 폼 필드만 편집 가능
- 컨트롤 요소에 집중

### 5. 인쇄 모드 (PRINT)
- 인쇄용 레이아웃
- 페이지 경계 표시

### 6. 설계 모드 (DESIGN)
- 문서 구조 편집
- 레이아웃 디자인

### 7. 주석 모드 (GRAFFITI)
- 문서에 주석/낙서 추가
- 손그림/메모 작성

## 목차 (Catalog)

### 기능
- 문서의 제목 구조를 트리 형태로 표시
- 제목, 편, 장, 절, 관, 조까지 계층 구조로 표시
- 항, 호, 목, 단은 목차에 표시되지 않음 (본문 들여쓰기만 적용)
- 항목 클릭 시 해당 위치로 자동 스크롤

### 표시/숨김
- 하단 좌측의 목차 아이콘 클릭
- 목차 패널 우측 상단 닫기(×) 버튼

## 주석 패널

### 기능
- 문서에 추가된 주석 목록 표시
- 주석 클릭 시 해당 위치로 이동
- 주석 삭제 (× 버튼)

### 주석 정보
- 선택 영역 텍스트
- 작성자 이름
- 작성 일시
- 주석 내용

## 키보드 탐색

### 커서 이동
- `↑` `↓` `←` `→`: 커서 이동
- `Home`: 줄 시작으로 이동
- `End`: 줄 끝으로 이동
- `Ctrl+Home`: 문서 시작으로 이동
- `Ctrl+End`: 문서 끝으로 이동
- `Page Up`: 한 페이지 위로
- `Page Down`: 한 페이지 아래로

### 텍스트 선택
- `Shift` + 방향키: 텍스트 선택
- `Ctrl+A`: 전체 선택

### 삭제
- `Backspace`: 이전 문자 삭제
- `Delete`: 다음 문자 삭제

## 팁과 트릭

### 💡 효율적인 사용법

1. **서식 복사 더블클릭**: 여러 곳에 같은 서식을 적용할 때 편리
2. **단축키 활용**: 자주 사용하는 기능은 단축키로 빠르게 접근
3. **목차 활용**: 긴 문서 작성 시 제목으로 구조화하고 목차로 빠르게 이동
4. **페이지 모드**:
   - 편집 시: 분할 모드
   - 읽기 시: 연속 모드
5. **검색 및 바꾸기**: 대량의 텍스트 수정 시 효율적
6. **컨트롤 활용**: 양식 문서 작성 시 유용
7. **워터마크**: 초안, 기밀 등 문서 상태 표시

### ⚠️ 주의사항

1. 저장 버튼은 서버로 데이터를 전송하므로 서버 설정 필요
2. 이미지는 Base64로 인코딩되어 저장되므로 큰 이미지는 성능 저하 가능
3. 컨트롤의 값 집합은 올바른 JSON 형식이어야 함
4. LaTeX 수식은 적절한 라이브러리 로드 필요
